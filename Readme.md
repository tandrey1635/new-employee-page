# Корпоративный портал для онбординга сотрудников "Авексима"
Внутренний веб-сайт для адаптации новых сотрудников, разработанный под Bitrix 24. Содержит всю необходимую информацию о компании, команде, правилах, графике, локациях и корпоративных предложениях. Включает портал обучения для сотрудников.

## 🛠 Технологический стек

### Frontend
- **HTML5** — семантическая верстка, БЭМ-методология
- **SASS/SCSS** — модульная структура, переменные, миксины, медиа-запросы
- **JavaScript** — интерактивные компоненты

## 🎯 Ключевые функции

### Пользовательский интерфейс
- Аккордеоны с контактами сотрудников по направлениям
- Аккордеоны Дрес-кода с визуальными примерами
- Сетка Аккордеонов с ближайшими организациями
- Карта офиса с легендой

### Интерактивные элементы
- Модальное окно с правилами коммуникации

### UX/UI компоненты
- Адаптивное бургер-меню для мобильных устройств
- Плавная навигация по странице

## 🔧 Технические особенности

### Адаптивность
- Адаптивный дизайн
- Медиа-запросы
- Кросс-браузерная совместимость


**Проект демонстрирует навыки верстки для корпоративных порталов и последующей интеграции в Bitrix 24.**
